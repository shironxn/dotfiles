# dotfiles

Arch Linux · Sway · Fish · Foot · Fuzzel · Neovim

## structure

```text
nvim/      text editor
sway/      window manager
fish/      shell
foot/      terminal
fuzzel/    launcher
lazygit/   git tui
fastfetch/ system info
yay/       aur helper
wal/       pywal colors
avizo/     notifications
pkg-lists/ package backups
```

## install

```bash
git clone https://github.com/shironxn/dotfiles.git ~/dotfiles
```

deploy manual via symlink or use `backup.sh`.
